## 3. 프로젝트 구조

```
└─AIRCLEAN-REACT
    ├─public
    ├─src
    │  ├─exception
    │  └─model
    │      ├─dto
    │      └─method
    ├─configuration
    │  └─handler
    ├─mainPage
    ├─mediConnect
    ├─mypage
    ├─staff
    ├─theraLink
    └─user
```

충돌을 방지하기 위해서 기능 별로 역할을 분담하고 기능별로 패키지를 구성하였습니다
